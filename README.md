# JSON格式化工具

> 双视图格式化显示JSON数据，是一款在线免费的json格式化工具，不需要登录，不需费用，用完即走，包有中文和英文版本。

## 功能特点

* 支持编辑框查看和树状分级查看
* 双视图显示
* 支持一键复制格式化结果

## 站点预览

* https://json.openai2025.com/

## 用到的库

* Bootstrap
* jQuery.js
* clipboard.js
* jquery.json-viewer
